## What I've Built

### LLM Brand Tracker — AI Brand Visibility Platform

> *Track how ChatGPT, Gemini, Claude, Grok, Google AI, and Perplexity rank your brand — free, automated, synced to Google Sheets*

- Drives real Chrome sessions via **Chrome DevTools Protocol (CDP)** — no API rate limits, no blocked responses
- Queries **6 AI platforms in parallel**, extracts brand rank and calculates **AI Share of Voice (SOV%)**
- Detects every competitor mentioned across platforms and maps their SOV% over time
- Auto-syncs to **5 structured Google Sheets tabs** after every run — no manual export
- Full-page screenshot captured from each platform per run
- Ships as a **Windows desktop app** (Electron + Node.js + React) — install and run, no server needed
- **Free** — every SaaS alternative charges $29–$800/month for less coverage
