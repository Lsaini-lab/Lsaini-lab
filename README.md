## 🧠 About

I specialize in **AI automation and LLM integration** — building tools at the intersection of browser automation, real-time data pipelines, and AI APIs.

My focus is **GEO (Generative Engine Optimization)** and **AEO (Answer Engine Optimization)** — measuring how brands appear inside AI-generated answers from ChatGPT, Gemini, Grok, and more. Almost no free tooling exists in this space, so I built some.

## 🚀 What I've Built

### ⚡ LLM Brand Tracker — AI Brand Visibility Platform

> *Track how ChatGPT, Gemini, Claude, Grok, Google AI & Perplexity rank your brand — automated, parallel, synced to Google Sheets*

- Drives real Chrome sessions via **Chrome DevTools Protocol (CDP)** — bypasses API limits and blocking
- Queries **6 AI platforms simultaneously**, extracts brand rank and calculates **AI Share of Voice (SOV%)**
- Maps every competitor mentioned across platforms and tracks SOV% over time
- Auto-syncs results to **5 structured Google Sheets tabs** per run — no manual export
- Full-page screenshot captured from each platform after every run
- Ships as a **native Windows desktop app** built with Electron + Node.js + React
- **Free** — SaaS competitors charge $29–$800/month for less coverage
